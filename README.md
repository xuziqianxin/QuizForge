# QuizForge

QuizForge 是一款基于命令行的题库管理与答题练习工具。它能够从学习平台导出的 HTML 作业/考试文件中提取题目，支持分类管理、随机抽题、答案对照与成绩统计，帮助您高效地复习和自测。

## ✨ 功能特性

- **智能解析**  
  自动识别 HTML 类型（作业详情页 / 考试详情页），精准提取题型、题干、选项及答案。  
  - 作业详情页：直接提取正确答案。  
  - 考试详情页：根据用户答题正误智能判定，答对则取用户答案为正确答案，答错则置空并提示题号，方便重点复习。

- **分类管理**  
  支持创建多级分类，每个分类下可独立存储多个题库文件（JSON 格式），便于按科目、章节组织题目。

- **批量导入**  
  支持导入单个 HTML 文件或整个文件夹，自动处理中文文件名转拼音，重名时可选择覆盖、跳过或全部覆盖。

- **灵活的答题模式**  
  - 随机打乱题目顺序与选项顺序（可选），增加练习挑战性。  
  - 支持答题过程中跳转（`j`）、返回跳转前位置（`j back`）、查看已答列表（`list`）、随时提交（`submit`）。  
  - 自动计算得分并输出对齐的答案对照表。

- **交卷后复盘**  
  交卷后可逐题浏览题干、你的答案、正确答案及正误，支持跳转、上一题/下一题导航，便于针对性复习。

- **一键启动脚本**  
  提供 Windows 批处理脚本，自动检测 Python 环境并安装依赖，双击即可运行。

## 📦 安装

### 环境要求
- Python 3.7 或更高版本

### 安装依赖
```bash
pip install beautifulsoup4
```
> 若希望中文文件名自动转拼音（推荐），可额外安装：
> ```bash
> pip install pypinyin
> ```

## 🚀 快速开始

1. 克隆或下载本项目到本地。
2. 将 `quizforge.py` 与 `start_quiz.bat` 放在同一目录。
3. 双击 `start_quiz.bat` 运行程序（或手动执行 `python quizforge.py`）。
4. 首次运行会自动创建 `question_banks` 文件夹，用于存放题库。

### 使用流程概览
- **导入题库**：主菜单选择 `1` → 创建/选择分类 → 选择 `从 HTML 导入`，输入 HTML 文件或文件夹路径。
- **开始答题**：主菜单选择 `2` → 选择分类 → 选择题库文件（可多选或全选） → 设置是否随机选项 → 答题。
- **查看结果**：交卷后查看得分，并可进入逐题复盘模式。
